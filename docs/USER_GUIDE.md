## 1. 시작하기

### 앱 실행

```bash
npm run tauri:dev    # 개발 모드
npm run tauri:build  # 프로덕션 빌드
```

### 화면 구성

앱은 세 영역으로 구성됩니다:

```
┌─────────────────────────────────────────────┐
│  타이틀바 (내비게이션 + 검색 + 뷰 전환)        │
├──────────┬──────────────────────────────────┤
│          │                                  │
│ 사이드바  │         메인 콘텐츠 영역           │
│          │                                  │
│ - 소스    │   (갤러리/중복/B컷/정리/동기화     │
│ - 라이브러리│    /리뷰/폴더트리/설정)           │
│ - 도구    │                                  │
│ - 통계    │                                  │
└──────────┴──────────────────────────────────┘
```

**타이틀바 탭:**
갤러리 | 폴더 | 중복 탐지 | 정리 | 동기화 | 히스토리 | 도구 ▾ | 설정 ⚙

**도구 드롭다운:**
- B컷 자동 탐지 - 선명도/노출 분석
- 사진 리뷰어 - 수동 B컷 마킹

## 2. 미디어 라이브러리

### 소스 폴더 추가

1. 사이드바 상단 **"폴더 추가"** 버튼 클릭
2. 폴더 선택 대화상자에서 사진/영상이 있는 폴더 선택
3. 자동으로 2단계 스캔 진행:
   - **Phase 0** (빠른 스캔): 파일 목록을 즉시 수집
   - **Phase 1** (상세 처리): EXIF 추출 + 썸네일 생성 (백그라운드)
4. 스캔 완료 후 갤러리에서 파일 확인 가능

> 여러 폴더를 추가할 수 있습니다. 각 폴더는 독립적으로 관리됩니다.

### 폴더 관리

- **리스캔**: 폴더 이름에 마우스를 올리면 나타나는 🔄 아이콘 클릭
- **폴더 제거**: 폴더 이름에 마우스를 올리면 나타나는 🗑 아이콘 클릭
- **라이브러리 초기화**: 사이드바 하단 "라이브러리 초기화" 버튼 (모든 데이터 삭제)

### 라이브러리 탐색

사이드바 **라이브러리** 섹션에서 파일을 필터링합니다:

| 항목 | 설명 |
|------|------|
| 모든 파일 | 전체 미디어 (사진 + 영상) |
| 사진만 | 이미지 파일만 표시 |
| 영상만 | 비디오 파일만 표시 |
| 날짜별 | 촬영일/수정일 기준 그룹핑 |
| 폴더별 | 원본 폴더 경로 기준 트리 탐색 |

## 3. 갤러리

미디어 파일을 탐색하고 미리보는 메인 뷰입니다.

### 보기 모드

타이틀바 우측의 토글 버튼으로 전환합니다:

**그리드 뷰 (기본)**
- 정사각형 썸네일 카드 형태
- 파일명 + 크기 표시
- 영상 파일은 재생 아이콘 표시

**리스트 뷰**
- 테이블 형태 (이름, 폴더, 촬영일, 수정일, 크기, 해상도, 유형)
- 컬럼 클릭으로 정렬 (▲/▼)
- 컬럼 경계 드래그로 너비 조절
- 행 밀도 선택: 좁게 / 보통 / 넓게

### 파일 미리보기

- 파일 클릭 → 미리보기 모달 열림
- **Space** 키로 선택된 파일 미리보기 토글
- 미리보기 모달에서:
  - **Quick Look**: macOS 네이티브 미리보기 (qlmanage)
  - **Finder에서 보기**: 파일 위치를 Finder에서 열기
  - 상세 정보: 유형, 크기, 해상도, 촬영일, 수정일, 경로

### 페이지네이션

- 초기 로드: 500개 (그룹핑 시 2,000개)
- 하단 **"더 보기"** 버튼으로 추가 로드

## 4. 중복 파일 탐지

동일한 파일을 찾아 정리합니다.

### 사용 방법

**1단계: 스캔**
- 타이틀바에서 **중복 탐지** 탭 클릭
- **"중복 스캔"** 버튼 클릭
- 3단계 분석 자동 진행:
  1. 파일 크기 분석 (100KB 이상)
  2. xxHash64 퀵 해시 비교 (4KB)
  3. SHA256 전체 해시 확인
- 진행률이 실시간으로 표시됩니다

**2단계: 그룹 검토**
- 좌측: 중복 그룹 목록 (그룹 번호, 파일 수, 일치 유형)
- 우측: 선택한 그룹의 멤버 상세 정보
  - 각 파일: 경로, 크기, 해상도, 촬영일
  - **"유지"** (녹색) 또는 **"삭제 예정"** (빨간색) 상태 표시

**3단계: 보관 파일 선택**
- 유지할 파일의 **"이것을 유지"** 버튼 클릭
- 나머지는 자동으로 "삭제 예정"으로 표시

**4단계: 삭제 실행**
- **삭제 미리보기**: 삭제될 파일 목록과 절약 용량 확인
- **"이 그룹 중복 삭제"**: 현재 그룹만 처리
- **"모든 중복 삭제"**: 전체 그룹 일괄 처리
- 삭제된 파일은 macOS 휴지통으로 이동 (복구 가능)

### 키보드 단축키

| 키 | 동작 |
|----|------|
| Space | 선택 그룹 미리보기 |
| ↑ / K | 이전 그룹 |
| ↓ / J | 다음 그룹 |

## 5. B컷 자동 탐지

연속 촬영된 사진 중 최고 품질을 자동 선택하고 나머지를 정리합니다.

### 사용 방법

**1단계: 설정**
- 타이틀바 **도구** → **B컷 자동 탐지** 클릭
- 상단에서 **시간 간격** 선택:
  - 2초 / 5초(기본) / 10초 / 30초 / 1분
  - 이 간격 내에 촬영된 사진은 같은 그룹으로 묶입니다

**2단계: 탐지 실행**
- **"분석 시작"** 버튼 클릭
- 2단계 진행:
  1. 시간 기반 그룹화 (EXIF 촬영일 분석)
  2. 품질 점수 계산 (선명도 45% + 노출 30% + 해상도 15% + 파일크기 10%)

**3단계: 그룹 검토**
- 좌측: 그룹 목록 (대표 썸네일, 멤버 수, 품질 점수)
- 우측: 멤버 카드 그리드
  - 👑 **BEST** 배지: 최고 품질 사진
  - **B컷** 라벨: 나머지 사진
  - 품질 점수 막대: 선명도(파랑), 노출(노랑), 종합 점수

**4단계: 최고 사진 변경 (선택)**
- 다른 사진을 최고로 지정하려면 **"베스트로 선택"** 클릭
- 또는 **1~9** 숫자 키로 빠르게 선택

**5단계: 정리**
- **삭제 미리보기**: 삭제될 B컷 목록 확인
- **"전체 B컷 정리"**: 모든 그룹의 B컷 일괄 삭제 (BEST 제외)

### 키보드 단축키

| 키 | 동작 |
|----|------|
| 1-9 | 해당 번호 멤버를 BEST로 선택 |
| S | 현재 그룹 건너뛰기 (무시) |
| Space | BEST 멤버 미리보기 |
| ↑ / K | 이전 그룹 |
| ↓ / J | 다음 그룹 |

## 6. 스마트 정리

파일을 자동으로 폴더 구조로 분류합니다.

### 사용 방법

**1단계: 설정**
- **정리** 탭 클릭
- **대상 폴더** 선택 (정리된 파일이 이동할 위치)
- **정리 방식** 선택:
  - **날짜별**: EXIF 촬영일 기준 → `2024-03-15/` 폴더 구조
  - **유형별**: 사진/영상 → `Images/`, `Videos/` 분류

**2단계: 미리보기**
- **"미리보기"** 버튼 클릭
- 이동 계획 표시:
  - 이동될 파일 수, 생성될 새 폴더 수, 총 크기
  - 파일별 이동 경로: `photo.jpg → /대상/2024-03-15/`

**3단계: 실행**
- 계획 확인 후 **"실행"** 버튼 클릭
- 진행률 표시 (현재 파일명 + X/Y)
- 완료 시 결과 요약: 이동 성공/실패 수

> 모든 이동 작업은 **히스토리**에 기록되어 되돌리기가 가능합니다.

## 7. 사진 리뷰어

폴더의 사진을 하나씩 검토하며 수동으로 정리합니다.

### 사용 방법

**1단계: 폴더 선택**
- 타이틀바 **도구** → **사진 리뷰어** 클릭
- **"폴더 선택하여 시작"** 버튼으로 검토할 폴더 선택

**2단계: 사진 검토**
- 화면 중앙에 사진이 크게 표시됩니다
- 상단 바: 현재 위치 (예: "15 / 127") + 보관/버리기 통계
- 하단 썸네일 스트립: 전체 사진 미니 미리보기

**3단계: 마킹**
각 사진에 대해 판단합니다:

| 동작 | 키 | 설명 |
|------|----|----|
| 보관 | **K** | ✅ 녹색 "유지" 배지 → 자동으로 다음 사진 |
| B컷 (버리기) | **Space** | ❌ 빨간 "B컷" 배지 → 자동으로 다음 사진 |
| 이전/다음 | **← →** 또는 **H L** | 사진 탐색 |
| 마킹 취소 | **Esc** | 현재 사진 마킹 해제 |
| 단축키 도움 | **?** | 단축키 목록 표시 |

**4단계: 정리**
- 상단의 **"정리하기"** 버튼 클릭
- B컷으로 마킹된 파일이 휴지통으로 이동
- 결과: "X개 파일 휴지통 이동 완료"

## 8. 폴더 분석

파일 시스템을 다양한 관점에서 분석하는 전체 화면 도구입니다.

### 4가지 모드 (탭으로 전환)

#### 폴더 트리
- 계층적 폴더 탐색 (최대 5단계)
- 파일 유형별 아이콘 표시
- 빵크럼(breadcrumb) 경로 탐색
- 파일 선택 후 복사/이동 가능

#### 용량 분석
- 폴더별 저장 공간 사용량 시각화
- 색상 코딩으로 대용량 폴더 식별
- 하위 폴더 드릴다운 탐색
- 최대 파일 20개 표시 (10MB 이상)

#### 분할 비교
- 두 폴더를 좌우로 나란히 표시
- 각각 독립적으로 탐색 가능
- 폴더 구조 비교에 유용

#### 날짜 폴더 정리
- `YYYYMMDD` → `YYYY-MM-DD` 형식으로 일괄 변환
- 미리보기 후 실행
- 충돌 시 병합 옵션 제공

## 9. 파일 동기화

소스 폴더에서 대상 폴더로 파일을 단방향 동기화합니다.

### 사용 방법

**1단계: 설정**
- **동기화** 탭 클릭
- **소스 폴더** 선택 (원본 위치)
- **대상 폴더** 선택 (백업/복사 위치)
- **옵션 설정**:
  - ☐ 체크섬 검증: xxHash64로 파일 내용 비교
  - ☐ 고아 파일 감지: 소스에 없는 대상 파일 표시
  - ☐ 동기화 후 자동 추출: 외장 디바이스 자동 해제
- **제외 패턴** (선택): 줄별로 glob 패턴 입력
  ```
  *.tmp
  .DS_Store
  Thumbs.db
  __MACOSX/**
  ```

**2단계: 미리보기 (Dry-Run)**
- **"미리보기"** 버튼 클릭
- 분석 결과 카드:
  - 🆕 새 파일: 대상에 없는 파일 (복사 예정)
  - 🔄 업데이트: 소스가 더 최근/크기 다른 파일
  - ⚠️ 충돌: 대상이 더 최근인 파일
  - 🗂 고아 파일: 소스에 없는 대상 파일

**3단계: 충돌 해결** (충돌이 있는 경우)
각 충돌 파일에 대해 선택:

| 옵션 | 설명 |
|------|------|
| **소스로 덮어쓰기** | 대상 파일을 소스로 교체 |
| **이름 변경 후 복사** | 대상을 .bak으로 이름 변경 후 소스 복사 |
| **건너뛰기** | 해당 파일 동기화 안 함 |

- **"모두 적용"** 버튼으로 동일 옵션 일괄 적용
- 내용이 동일한 경우 "내용 동일" 배지 표시 (안전하게 건너뛸 수 있음)

**4단계: 동기화 실행**
- **"동기화 시작"** 버튼 클릭
- 진행률 표시: 파일 수 + 바이트 전송량
- **"취소"** 버튼으로 중단 가능

**5단계: 결과 확인**
- 복사/업데이트/건너뜀 수
- 총 전송량
- 오류 목록 (있는 경우)

### 프리셋

자주 쓰는 동기화 설정을 저장하여 재사용합니다:

- 우측 패널에서 **프리셋 이름** 입력 → **"저장"**
- 저장된 프리셋 클릭 → 설정 자동 로드
- 프리셋 삭제: 호버 시 🗑 아이콘

## 10. 외장 디바이스

SD카드, USB 드라이브 등 외장 디바이스를 자동으로 감지합니다.

### 동작 방식

- 사이드바 하단에 연결된 외장 디바이스 자동 표시
- 3초 간격으로 마운트/언마운트 폴링
- 각 디바이스 표시 정보:
  - 아이콘 (USB/HDD)
  - 볼륨 이름 + 마운트 경로
  - 용량 바 (사용량 비율 색상 표시: 초록/노랑/빨강)
  - 여유 공간 / 전체 용량

### 사용 가능한 동작

| 동작 | 설명 |
|------|------|
| **소스로 추가** | 디바이스를 소스 폴더로 등록하여 스캔 |
| **추출** | 안전하게 디바이스 해제 (diskutil eject) |

## 12. 작업 히스토리 & 되돌리기

모든 파일 이동/삭제 작업은 자동으로 기록됩니다.

### 히스토리 확인

- **히스토리** 탭 클릭
- 배치별로 그룹핑된 작업 목록:
  - 배치 이름, 작업 수, 실행 시각
  - 펼치기 → 개별 작업 상세 (순서, 작업 유형, 소스/대상 경로)

### 되돌리기

1. 되돌리고 싶은 배치의 **"되돌리기"** 버튼 클릭
2. 해당 배치의 모든 작업이 역순으로 취소됩니다
3. 이동된 파일은 원래 위치로 복원

> 되돌리기는 가장 최근 배치부터 순서대로 하는 것을 권장합니다.

## 일반 사용 팁

1. **처음 사용 시**: 폴더 추가 → Phase 1 완료 대기 → 중복 탐지 → B컷 탐지 → 정리
2. **SD카드 워크플로우**: SD카드 삽입 → 자동 감지 → 소스로 추가 → 스캔 → 동기화 → 자동 추출
3. **정기 관리**: 스케줄 설정으로 매일 자동 스캔 + 중복 탐지 예약
4. **안전한 삭제**: 항상 "미리보기"로 확인 후 삭제 → 휴지통에서 최종 복구 가능
5. **대량 파일**: 큰 폴더는 Phase 1이 오래 걸릴 수 있음 → 백그라운드에서 자동 진행
6. **외국어 UI**: 설정 → 일반 → 언어에서 English로 전환 가능
